# markdown format

<pre><code><strong># h1
</strong>## h2
### h3
#### h4
##### h5
###### h6

- bullet list
  - list item

1. ordered list
  1. list item

**bold**

[link-text](link-href)

`inline code`
</code></pre>

일반 단축키

![](<../../../.gitbook/assets/CleanShot 2025-05-13 at 14.29.40.png>)\\

* Cmd+Z / Cmd+Shift+Z (undo / redo)
* Cmd + C / Cmd + V
* List item에서
  * tab / shift tab (sink, lift list item)
  * cmd + delete (리스트 아이템의 기호를 유지한채로 삭제)
* 드래그 후
  * Cmd + B (bold)
  * Cmd + E (inline code)
* paragraph에서 Shift + Enter => Line break (문단 구분 없는 엔터)

비밀 단축키\\

* Mod+Shift+8
* Mod+Shift+7
* 연속으로 아무곳에서나 `#debugmode` 입력 > 마크다운 렌더 프리뷰

직접 입력을 권장하지 않는 문법\\

```
Pin (Anchor)
^AA^

Go to pin link
[Go to #AA](#AA)

Tool tags
{{tool:tool-id:tool-name}}
```
