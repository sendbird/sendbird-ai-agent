# Actionbooks

The Actionbooks feature lets you create instructions that guide AI agents in handling tasks and scenarios. The main advantage is that flows and tasks are written in natural language, making them accessible to all stakeholders.

**Tips on writing step-by-step instructions**

Use Enter and Tab to organize your instructions that are like conversation trees:

* Enter creates different response paths
* Tab adds next steps with a response path

Below is an example of a step-by-step instruction for a cancellation flow:

<figure><img src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXdKIDAyKgnWHZhsVU2vLsKiKP7lGzDmBBcI9qYoMUeIJ-7fFfVWVHRjeKZLQ_alMqUj0CCjgg2kgxvwJcR8BvlgDfmEGmAO1OBx2QMTX9-If1EujqyZhUEMHny_-jQRjLcBkZSc8Q?key=xS65XOd58rPJOYVnOh8lJEX0" alt=""><figcaption></figcaption></figure>

In this case,

1. The AI agent asks why the customer wants to cancel
2. Based on their response, the agent follows the relevant path:
   1. If they mention high prices (1a)
   2. If they simply don’t want the plan anymore (1b)
   3. If they experienced delivery issues (1c)
3. The agent provides the appropriate response and continues the conversation until reaching resolution.
