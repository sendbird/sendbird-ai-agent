# Build

The **Build** menu allows you to customize your AI agent’s appearance, behavior, and customer interactions. You can configure the following features:\\

* [**Profile**](profile.md): Set your AI agent’s name and profile image, and define its behavior guidelines and communication style.
* [**Knowledge**](../shared-assets/knowledge.md): View the detailed information of **Knowledge** articles that the current AI agent is using when assisting users. To update or add more data in **Knowledge**, view our [guide on Shared assets > Knowledge](../shared-assets/knowledge.md).
* [**Actionbooks**](../shared-assets/actionbooks/): View the detailed information of **Actionbook** flows that the current AI agent is using when assisting users. To update or add more work flow of **Actionbooks**, view our [guide on Shared assets > Actionbooks](../shared-assets/actionbooks/).
* [**CSAT**](csat.md): Customize your customer satisfaction (CSAT) survey and collect user feedback on your AI agent's performance. You can add follow-up questions and even support multi language via auto-translation.
* [**Safeguards**](safeguards.md): Control your AI agent responses by setting protective barriers and defining how it handles potential misuse. Add content filters, block specific words, and write custom messages that display when safety limits are reached.
* [**Channels**](channels/): Configure and manage the settings of multi channels in which your AI agent can communicate with users. Sendbird AI agent can assist users via the following channels:
  * [**Messenger**](channels/messenger/): an in-app native conversation channels
    * **Messenger** > [**Conversation settings**](channels/messenger/conversation-settings.md): Define how an AI agent interacts with customers, including the welcome message, human-agent handoff options, and conversation timeout rules.
    * **Messenger** > [**Appearance**](channels/messenger/appearance.md): Customize the style and appearance of your AI agent widget by configuring the theme, colors, icons, and related settings.
  * [**Voice**](channels/voice.md): a voice all channel that enables your AI agent to talk with users over the phone.
  * [**SMS**](channels/sms.md)
  * [**Email**](channels/email.md)
  * [**WhatsApp**](channels/whatsapp.md)
