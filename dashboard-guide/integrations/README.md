# Integrations

In the **Integrations** menu, you can manage the integration of customer support software to enable features such as automated ticket creation and seamless hand-off to human agents. This ensures a unified, single-channel experience that facilitates smooth transitions between AI and human agents. This also guarantees uninterrupted and efficient support for your end-users. You can also directly integrate external knowledge sources so that the AI Agent can retrieve relevant information during user interactions.

***

## Platforms

We support integrations with tools like Confluence, Salesforce, and ZenDesk allowing you to maximize the value of your existing knowledge sources.

* [**Zendesk**](zendesk.md)
* [**Salesforce**](salesforce.md)
* [**Freshworks**](freshworks.md)
* [**Confluence**](confluence.md)
* [**Sendbird Desk**](sendbird-desk/)
* [**Sprinklr**](sprinklr.md)
* [**Amazon S3**](amazon-s3.md)
* [**Genesys**](genesys.md) (Voice only)
